SimConnect client

SimConnect client is a local application that connects to the SimConnect API of Microsoft Flight Simulator.

It allows you to access the data from the simulator and send commands to it.

To use it, you need to have Microsoft Flight Simulator installed and running.

Remote connect

If you want to connect to a remote SimConnect client, you need to enable your personal hotspot to connect to the same network as your computer.

Then, check the checkbox in SimConnectMap to enable remote connection. It will automatically scan the IP's for the active simulator and connect to it.
